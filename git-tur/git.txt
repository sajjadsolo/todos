14-git stash
git stash
git stash list (show all stash)
git stash save "save commmond stash"
git stash save -u (untracked file)
git stash save --include-untracked (untracked file)
git stash -a (ignored file saved)
git stash apply (restore save stash)
git stash apply stash@{id}
git stash pop (restore file saved but delete last stash is removed)
git stash pop stash@{1}
git stash drop (delete stash)
git stash drop stash@{1}
git stash clear (remove all stash and not restore)
git stah show (show change stash)
git stash show stash@{1}
git stash show -p (change all stash)
git stash branch <name> (build branch and switch and delete stash)
